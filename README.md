# the-replay

THE REPLAY — deterministic communication proven. Two looms, one seed: the op-streams diff to exactly 0, byte-for-byte, live. Flip in one grain of entropy and the guarantee dies (the diff lights red). Determinism measured, not asserted — the axiom under the corpus's verify-not-assert ethos.

**Live:** https://davidwise01.github.io/the-replay/
